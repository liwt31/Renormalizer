class renormalizer.utils.configs.CompressConfig(criteria: renormalizer.utils.configs.CompressCriteria = <CompressCriteria.threshold: 'threshold'>, threshold: float = 0.001, bonddim_distri: renormalizer.utils.configs.BondDimDistri = <BondDimDistri.uniform: 'uniform'>, max_bonddim: int = 32)[source]¶

Bases: object

MPS Compress Configuration.

Parameters

  • criteria (CompressCriteria) – the criteria for compression.

  • threshold (float) – the threshold to keep states if criteria is set to CompressCriteria.threshold or CompressCriteria.both.

  • bonddim_distri (BondDimDistri) – Bond dimension distribution if criteria is set to CompressCriteria.fixed or CompressCriteria.both.

  • max_bonddim (int) – Maximum bond dimension under various bond dimension distributions.

class renormalizer.utils.configs.CompressCriteria[source]¶

Bases: enum.Enum

Criteria for compression.

both = 'both'¶

Compress combining threshold and fixed. The bond dimension for the two criteria are both calculated and the smaller one is used.

fixed = 'fixed'¶

Compress depending on pre-set fixed bond dimension.

threshold = 'threshold'¶

Compress depending on pre-set threshold. States with singular value smaller than the threshold are discarded.
